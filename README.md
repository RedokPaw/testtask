### Build and run
Использовать run.cmd для сборки и запуска

### Usage
Сервер доступен по адресу **localhost:8080**

Swagger расположен по дефолт адресу: **localhost:8080/swagger-ui/index.html**

### Description
TL;DR Для сортировки был выбран алгоритм Quickselect из-за его хорошей средней сложности O(n)

Cчитаю, что для данной задачи подошёл бы еще способ с использованием min-heap из-за готовой реализации 
в Java PriorityQueue. Так же есть вариант с median of medians, но разница при небольших данных не такая большая.
